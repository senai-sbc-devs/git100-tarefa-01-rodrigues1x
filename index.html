Git Workflow

Um guia visual para o fluxo de trabalho do Git.

Clone

Clone um repositório remoto para o seu computador.

git clone [URL]

Branch

Crie ramos para desenvolver funcionalidades isoladamente.

git branch [nome-do-ramo]

Commit

Registre as alterações no repositório.

git commit -m "Mensagem"

Merge

Combine ramos para integrar as alterações.

git merge [nome-do-ramo]

Pull Request

Solicite a integração das alterações no repositório principal.

git pull-request
